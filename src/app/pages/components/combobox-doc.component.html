Combobox

gc-combobox

Selector con búsqueda, modo single o multiple, y tags.

API

InputTypeDefaultDescripción
mode'single' | 'multiple''single'Modo de selección
options(string | GcComboboxOption)[][]Lista de opciones
valuestring | nullnullValor seleccionado (single)
valuesstring[][]Valores seleccionados (multiple)
placeholderstring''Placeholder del input
disabledbooleanfalseDeshabilita el combobox
invalidbooleanfalseMarca como inválido
emptyTextstring—Texto cuando no hay resultados
colorstring | nullnullColor de focus
